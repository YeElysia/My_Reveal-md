第三章: 人工智能的应用开发基础 (详细资料)

3.1 Python 基础入门

Python是目前人工智能领域最主流的编程语言，因其语法简洁、社区生态丰富而备受青睐。掌握其基础是进行AI开发的前提。

[此处应插入图片：Python数据类型与变量思维导图，见课件第 179 页]

核心数据类型

  • 数字 (Number)：包括整数(int)、浮点数(float)、复数(complex)。
  • 布尔型 (Boolean)：只有两个值，`True` 和 `False`。
  • 字符串 (String)：用单引号或双引号括起来的文本序列。
  • 列表 (List)：用方括号 `[]` 表示，是有序、可变的集合。可以包含不同类型的元素。
  • 元组 (Tuple)：用圆括号 `()` 表示，是有序、不可变的集合。一旦创建就不能修改。
  • 集合 (Set)：用花括号 `{}` 表示，是无序、不重复的元素集合。主要用于去重和成员测试。
  • 字典 (Dictionary)：用花括号 `{}` 表示的键值对（key-value）集合。键必须是唯一的。

程序控制结构

  • 顺序结构：代码从上到下依次执行。
  • 选择结构：使用 `if`, `elif`, `else` 语句根据条件判断执行不同的代码块。
  • 循环结构：
    • `for` 循环：用于遍历一个序列（如列表、字符串）或其他可迭代对象。
    • `while` 循环：当给定条件为真时，重复执行代码块。

3.2 AI开发环境与工具包

一个稳定、配置良好的开发环境是高效进行AI项目开发的基础。

集成开发环境 (IDE)

  • Anaconda：一个开源的Python发行版本，集成了大量科学计算、数据分析的包。其优点是集成度高，可视化好，方便管理包和环境。但平台内许多高级工具可能需要额外收费。
  • PyCharm：一款功能强大的Python IDE。社区版免费，专业版收费。其优点是可以直接将项目文件夹映射为虚拟环境，并支持远程开发，功能强大。
  • Jupyter Notebook/Lab：一种基于Web的交互式计算环境，允许创建和共享包含代码、公式、可视化和叙述性文本的文档。非常适合数据探索和教学演示。

虚拟环境

作用：解决不同项目间Python版本或第三方库版本的兼容性问题和冲突。为每个项目创建一个独立的、干净的虚拟开发环境，是AI开发的最佳实践。

包管理工具：pip

`pip` 是Python的包安装程序，用于安装和管理第三方库。

# 使用国内镜像源（如清华源）加速安装
pip install numpy -i https://pypi.tuna.tsinghua.edu.cn/simple

核心AI算法库

  • NumPy：Python科学计算的基础包，提供了强大的N维数组对象和相关运算功能。
  • Pandas：提供了高性能、易于使用的数据结构（如DataFrame）和数据分析工具。
  • Matplotlib：一个强大的2D绘图库，用于数据可视化。
  • Scikit-learn：一个功能丰富的机器学习库，包含了大量的分类、回归、聚类和降维算法，以及模型选择和评估工具。

3.3 深度学习框架

深度学习框架提供了一套高级的API和工具，极大地简化了构建、训练和部署复杂神经网络的过程。

  • TensorFlow：由Google开发。特点是生态系统完善，支持工业级部署（通过TensorFlow Serving），并且拥有强大的可视化工具TensorBoard。其高级API `Keras` 以用户友好和快速原型设计而闻名。
  • PyTorch：由Facebook（Meta）开发。特点是其动态计算图机制，使得模型构建和调试更加灵活直观，深受学术界和研究人员的喜爱。语法风格与原生Python更接近。
  • 国产框架：如百度的飞桨(PaddlePaddle)、华为的MindSpore等也在迅速发展，构成了自主可控的AI生态。
